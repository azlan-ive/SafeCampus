**SafeCampus** is a crowdsourced safety application designed to help students stay aware of their surroundings on campus. It allows users to report incidents and view real-time safety information using an interactive map interface, helping the community make safer decisions in real time.

## ğŸš€ Features

- **Real-Time Map**: View campus incidents and static "Admin Marked" locations (Security Posts, Clinics, Emergency Points) on a Google Map.
- **Incident Reporting**: Easy-to-use form to report accidents, crimes, medical emergencies, and fires.
- **Dynamic Tracking**: Automatically detects your current location (e.g., UiTM Jasin) and translates coordinates to a readable address.
- **User Accountability**: Every report stores the Name and User ID of the reporter.
- **Admin Marked Locations**: A dedicated list of safety points managed by campus administrators.
- **User Profiles**: Manage personal account information fetched directly from Firebase.
